# waypoint

## 문제 설명

> 다수의 드론 비행 기록(CSV)이 주어진다.

- 주어진 파일: `DJIFlightRecord_2024-06-12_*.csv` 형식의 파일

## 풀이

### 분석

파일을 하나 열어보면 DJI 드론의 비행 로그라는 것을 알 수 있다. 핵심 컬럼은 위도와 경도이며, 시간 순서대로 드론의 위치가 기록되어 있다.

좌표를 보면 거의 한 점인 매우 좁은 구역에 점들이 몰려 있다. 또한 문제 힌트로 봤을때 위도와 경도에 무언가 정보가 숨겨져 있다는 것을 추측할 수 있고, 그 데이터를 약 50개 정도의 파일로 쪼개놨다는 것을 알 수 있었다.

### 취약점

-

### 익스플로잇

엑셀의 Power Query 기능으로 51개 CSV를 한 번에 합친 뒤, 합쳐진 데이터의 경도/위도를 분산형(scatter) 그래프로 그렸다.

1. 모든 CSV 파일을 한 폴더에 모은다.
2. 엑셀 Power Query 기능으로 모든 파일을 하나의 테이블로 병합한다.
4. 병합된 테이블에서 longitude(X축), latitude(Y축) 두 열을 선택한다.
5. 분산형 차트로 그래프를 그린다.
6. 점들이 모여 그려진 글자를 읽으면 플래그가 나타난다.

분산형 그래프 위에 비행 경로가 글자 형태로 나타나며, 이를 그대로 읽어 플래그를 획득했다.

## 배운 점

- 엑셀 Power Query의 폴더에서 가져오기 + 결합 기능은 동일한 스키마의 파일 수십 개를 코드 없이 한 번에 합칠 수 있어 데이터 정리에 매우 유용하다.
- 좌표 데이터를 그릴 때는 축 설정(경도=X, 위도=Y)에 주의해야 의도한 방향으로 그림이 보인다.
